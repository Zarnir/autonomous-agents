# AGENTS.md

This project uses the autonomous-agents spec-driven development pipeline.

**For all AI coding/planning tools (Cursor, Cline, Aider, Gemini CLI, Codex, Copilot, etc.):** read `CLAUDE.md` in this directory. The instructions there apply to you too. The two files are kept in sync — `AGENTS.md` exists as a discovery alias for tools that prefer this filename.

## Quick reference

- **Specs go in:** `docs/specs/epics/NN-name.md`
- **Format spec:** `docs/specs/AUTHORING_GUIDE.md`
- **Working example:** `docs/specs/EXAMPLE.md`
- **Validate before claiming done:** `aa-orchestrator validate`
- **Don't run** `aa-orchestrator develop` without explicit user instruction
